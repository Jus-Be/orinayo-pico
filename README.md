# Orinayo
A wireless USB MIDI controller for the CRKD-Gibson Guitar powered by a Raspberry Pico 2_W device

https://github.com/user-attachments/assets/bf5d0bad-dd6a-400b-bc92-5dadf5c17ba5

## Features

- **USB MIDI Output**: Send MIDI messages via USB and UART
- **USB Audio Output**: Stereo audio output at 48kHz (NEW!)
- **Bluetooth Connectivity**: Wireless control via Bluetooth
- **Step Sequencer**: Built-in sequencer for patterns
- **Multi-synthesizer Support**: Control for Yamaha, Ketron, and other devices

## USB Audio Output

The device now supports dual USB functionality:
- **USB MIDI Device**: Full MIDI controller with input/output
- **USB Audio Device**: Stereo 48kHz 16-bit audio output

See [USB Audio Output Documentation](docs/USB_AUDIO_OUTPUT.md) for detailed usage instructions.
