## Core Hardware Platform

### Raspberry Pi Pico 2 W

The system is specifically designed for the **Raspberry Pi Pico 2 W** board. This board features the RP2350 microcontroller and the CYW43439 wireless chip.

| Component | Specification | Usage in System |
| --- | --- | --- |
| **Microcontroller** | RP2350 dual-core ARM Cortex-M33 | Primary processing, MIDI routing, and chord generation. |
| **Wireless Chip** | Infineon CYW43439 | Bluetooth Classic (HID) and BLE MIDI connectivity. |
| **Flash Memory** | 2MB+ | Firmware storage and persistent looper pattern storage. |
| **RAM** | 520KB SRAM | Runtime operation and sequence buffers. |

The build system explicitly targets the `pico2_w` board variant with the `rp2350-arm-s` platform [README.md L62](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L62-L62)

### CYW43 Wireless Configuration

The system utilizes the CYW43439 chip for both Bluetooth Classic and BLE. The Bluetooth stack is configured to support a limited number of simultaneous connections to ensure stability and avoid shared bus overruns with the CYW43 chip [btstack_config.h L61-L64](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/btstack_config.h#L61-L64)

## Bluetooth Input Devices

Orinayo acts as a gateway for several types of wireless controllers. The system is optimized for low-latency HID and BLE MIDI processing.

### Supported Device Types

| Device Category | Protocol | Examples |
| --- | --- | --- |
| **HID Gamepads** | Bluetooth Classic | Guitar Hero-style controllers (CRKD-Gibson), Generic HID gamepads [README.md L28-L29](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L28-L29) |
| **BLE MIDI** | BLE | Liberlive, Sonic Master, Generic BLE MIDI Controllers [README.md L30-L31](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L30-L31) |

### Bluetooth Stack Configuration

The stack is configured in `sdkconfig.h` and `btstack_config.h` with the following hardware-relevant limits:

* **Max Devices**: Limited to 1 device by default to ensure maximum performance for the primary controller [sdkconfig.h L8](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/sdkconfig.h#L8-L8)
* **Security**: GAP Security is enabled for pairing [sdkconfig.h L10](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/sdkconfig.h#L10-L10)
* **Protocol Support**: Both BLE and Bluetooth Classic are enabled [btstack_config.h L13-L30](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/btstack_config.h#L13-L30)

## MIDI Output Interfaces

The system provides simultaneous dual MIDI output through USB and hardware UART.

### USB MIDI Output

The Pico 2 W acts as a **USB MIDI Client** (connecting to a DAW) and supports **USB MIDI Host** mode (connecting to other MIDI compliant devices) [README.md L36-L37](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L36-L37)

| Interface | Connector | Usage |
| --- | --- | --- |
| **USB Client** | USB-C/Micro-B | Connection to Computer/DAW |
| **USB Host** | USB-A (via adapter) | Connection to Class-Compliant Synths |

### UART MIDI Output

For connection to traditional hardware synthesizers or sound modules (like the M5Stack SAM2695), the system uses the hardware UART0 [README.md L38](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L38-L38)

**UART Pin Assignment**

| Signal | GPIO Pin | Physical Pin | Function |
| --- | --- | --- | --- |
| **TX** | GPIO 0 | Pin 1 | MIDI Transmit (31,250 baud) |
| **RX** | GPIO 1 | Pin 2 | MIDI Receive |

## Summary Hardware Checklist

### Required Components

* **Microcontroller**: Raspberry Pi Pico 2 W (RP2350 based) [README.md L62](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L62-L62)
* **Power/Data**: USB cable for power and MIDI communication.
* **Input**: A supported Bluetooth Classic HID gamepad or BLE MIDI device [README.md L65](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L65-L65)

### Optional Components

* **Hardware MIDI**: 5-pin DIN connector or M5Stack SAM2695 for hardware UART MIDI [README.md L38](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L38-L38)
* **External Synth**: Devices like Yamaha SeqTrak, MODX, or SP404 Mk2 for specific operational modes [README.md L48-L56](https://github.com/Jus-Be/orinayo-pico/blob/6dde5a75/README.md?plain=1#L48-L56)
